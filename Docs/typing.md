幫我建立一個白板、純前端適合競程使用白板網頁，其中，內容請包括：
1. 元素：資料結構（ 矩陣 ( 最多 2 維 )、堆疊、佇列等）、 樹 ( 紅黑樹、平衡樹... 可以供使用者選擇，並且支援文字生樹，例如：
```
節點數
節點值 父節點值
...
```
這樣的形式)
圖論 ( 有向圖、無向圖，一樣要支援文字生圖 )
```
節點數 邊樹
u v
...
```
、基本圖案 ( 正方形 圓形 等等)

2. 風格：深灰色主題，格線為淺灰色（ 不搶眼 ）。
3. 大小：可自由縮放大小

4. 色彩：介面大多請以黑白為主

5. 特殊要求： 文字、圖形、線段可以調整顏色、透明度、飽和度，另外提供 16 種基本顏色供挑選，不要太搶眼，圖論、樹那些的要可以自由拉取節點、線段、連接兩節點等等。

自由縮放：整個畫布可放大/縮小，支援滑鼠滾輪縮放。

拖拉 & 調整：所有元素可自由移動、調整大小、旋轉。

圖層管理：可選擇元素前後順序，方便遮擋處理。

多選與群組操作：選取多個元素，一起拖拉或更改顏色。
